# Examples

这里放各阶段的可运行示例代码。

建议每个示例都保持最小可运行：

- 一个 README 说明怎么运行
- 一个入口文件
- 清晰的依赖说明
- 不提交真实 API Key

## 示例规划

1. `01-prompt`：Prompt 模板示例
2. `02-llm-api`：命令行聊天助手
3. `03-tool-use`：Function Calling 示例
4. `04-rag`：本地文档问答示例
5. `05-agent`：最小 ReAct Agent
6. `06-mcp`：自定义 MCP Server
7. `07-claude-code`：Claude Code 工作流示例
