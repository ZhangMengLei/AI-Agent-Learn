# 03 Tool Use / Function Calling

## 学习目标

让模型不仅能回答问题，还能根据任务调用外部工具完成动作。

## 核心概念

- Tool Use
- Function Calling
- JSON Schema
- 工具选择
- 参数生成
- 工具执行结果回传
- 权限确认
- 工具调用安全

## 常见工具

- 搜索工具
- 文件读取工具
- 数据库查询工具
- HTTP API 工具
- 代码执行工具
- 浏览器工具

## 学习任务

1. 定义一个天气查询工具。
2. 定义一个计算器工具。
3. 定义一个本地文件读取工具。
4. 让模型根据用户问题自动选择工具。
5. 给高风险工具加入人工确认。

## 实战项目

做一个 `工具调用助手`：

- 能判断是否需要调用工具
- 能生成合法参数
- 能读取工具结果继续回答
- 能处理工具错误

## 安全重点

- 不要让模型直接执行任意命令。
- 用户输入进入 shell、SQL、文件路径时必须谨慎。
- 工具权限要最小化。
- 工具结果可能包含 Prompt Injection。
