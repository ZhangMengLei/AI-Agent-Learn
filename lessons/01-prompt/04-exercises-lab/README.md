# Prompt 练习工程说明

本目录用于完成 Prompt Engineering 阶段的练习实验。它更像一个“文本实验台”，重点不是写可运行代码，而是通过多轮编写、测试、对比和复盘，理解 Prompt 如何影响模型输出。

## 练习目标

完成本实验后，你应该能够：

1. 把一个模糊任务改写成清晰 Prompt。
2. 使用角色、任务、背景、约束、示例和输出格式控制结果。
3. 对同一个任务设计多个 Prompt，并比较输出差异。
4. 识别模型编造、格式不稳定、遗漏约束等常见问题。
5. 形成一套可复用的 Prompt 测试记录方法。

## 推荐目录

建议在本目录下按下面结构整理你的练习成果：

```text
03-exercises-lab/
```

说明：

- `exercise-*.md`：你自己的练习过程和实验记录。

- 如果只是课堂练习，也可以只在一个 Markdown 文件中记录全部实验。

## 运行 / 使用方式

本阶段不要求编写程序。你可以使用任意大模型产品或本地模型完成实验，例如：

1. 打开一个 AI 对话产品。
2. 复制练习中的 Prompt。
3. 输入测试文本或测试任务。
4. 保存模型输出。
5. 记录“Prompt 写法”和“输出效果”。
6. 根据问题修改 Prompt，再重复测试。

建议每个实验至少记录 3 次结果：

```text
实验编号：
任务目标：
Prompt 版本：
测试输入：
模型输出摘要：
发现的问题：
下一版改进：
```

## 练习 1：文章总结 Prompt

### 任务

设计一个 Prompt，让模型总结一篇文章，并输出：

```text
一句话总结：
核心要点：
适合谁看：
行动建议：
不确定信息：
```

### 实验要求

- 明确说明“只能基于原文总结”。
- 面向初学者，用中文回答。
- 如果原文没有提到，不要补充事实。
- 至少测试一篇技术文章和一篇非技术文章。

### 记录重点

观察模型是否会：

- 增加原文没有的信息。
- 把“行动建议”写得过于泛泛。
- 漏掉“适合谁看”。
- 输出格式不稳定。

## 练习 2：代码解释 Prompt

### 任务

设计一个 Prompt，让模型给新手解释一段代码。

### 输出格式建议

```text
代码整体作用：
执行流程：
关键变量：
易错点：
可以如何改进：
给新手的一句话总结：
```

### 实验要求

- 不要求模型重写全部代码。
- 要求模型先解释目的，再解释细节。
- 对不确定的依赖、上下文或业务含义，要说明“不确定”。

## 练习 3：JSON 信息提取 Prompt

### 任务

从用户文本中提取结构化信息。

目标 JSON：

```json
{
  "name": null,
  "job": null,
  "goal": null,
  "keywords": []
}
```

### 实验要求

- 严格输出 JSON。
- 不输出 Markdown 代码块。
- 缺失字段使用 `null`。
- `keywords` 必须是字符串数组。
- 至少测试一条信息完整输入、一条信息缺失输入、一条干扰信息较多输入。

## 练习 4：Prompt 对比实验

### 任务

对同一个任务分别设计 3 个 Prompt：

1. 简单 Prompt：只描述任务。
2. 增强 Prompt：加入角色和背景。
3. 稳定 Prompt：加入角色、任务、背景、约束、输出格式和示例。

### 对比表

```text
任务：
测试输入：

Prompt 1：
输出 1：
问题：

Prompt 2：
输出 2：
问题：

Prompt 3：
输出 3：
问题：

我的结论：
```

## 练习 5：Prompt Chaining

### 任务

把一个复杂任务拆成多个连续 Prompt，例如“从需求到测试用例”：

1. 提取需求目标。
2. 拆解功能点。
3. 识别边界情况。
4. 生成测试用例表格。
5. 检查遗漏。

### 实验要求

- 每一步只做一类事情。
- 下一步 Prompt 要引用上一步输出。
- 记录“拆成多步”与“一次性完成”的差异。

## 参考答案说明

参考答案不是唯一正确答案。判断 Prompt 好坏，重点看它是否：

1. 任务清晰。
2. 上下文充分。
3. 约束明确。
4. 输出格式稳定。
5. 对缺失信息不编造。
6. 可以被别人复用。

如果参考答案和你的答案不同，不代表你的答案一定错误。你需要通过测试输出证明它是否有效。

## 完成标准

完成本实验至少需要提交或保留以下内容：

- 5 个练习的 Prompt 文本。
- 每个练习至少 2 组测试输入和输出记录。
- 每个练习至少 1 条改进说明。
- 一段 200 字以内的复盘：你认为 Prompt 最重要的 3 个要素是什么。
