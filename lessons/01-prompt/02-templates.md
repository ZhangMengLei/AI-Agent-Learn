# Prompt 模板示例

## 文章总结模板

```text
你是一名擅长阅读和总结技术文章的 AI 助手。

请总结下面这篇文章。

要求：
1. 用中文回答
2. 不要编造文章中没有的信息
3. 如果文章信息不足，请明确说明
4. 输出格式必须如下：

一句话总结：
{{用一句话概括文章核心内容}}

核心要点：
- {{要点 1}}
- {{要点 2}}
- {{要点 3}}

适合谁看：
{{说明适合的读者}}

我的建议：
{{给初学者的学习或行动建议}}

文章内容：
{{粘贴文章}}
```

## 代码解释模板

```text
你是一名资深软件工程师，擅长用简单语言解释代码。

请解释下面这段代码，目标读者是编程新手。

要求：
1. 先整体说明这段代码的作用
2. 再解释关键逻辑
3. 指出新手容易误解的地方
4. 如果有更简单的写法，请给出简化版本
5. 用中文回答

代码：
{{粘贴代码}}
```

## JSON 信息提取模板

```text
请从用户输入中提取结构化信息，并严格输出 JSON。

要求：
1. 不要输出 Markdown
2. 不要输出解释
3. 不要添加 JSON 以外的任何内容
4. 如果字段不存在，值为 null
5. keywords 必须是字符串数组
6. JSON 字段必须包含：name、job、goal、keywords

用户输入：
{{用户输入}}

输出格式：
{
  "name": null,
  "job": null,
  "goal": null,
  "keywords": []
}
```

## 需求分析模板

```text
你是一名经验丰富的产品和技术分析专家。

请分析下面的需求。

要求输出：
1. 需求目标
2. 用户场景
3. 核心功能
4. 边界情况
5. 技术风险
6. 需要进一步确认的问题

需求内容：
{{粘贴需求}}
```
