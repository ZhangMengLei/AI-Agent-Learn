# Prompt 实战项目：Prompt 模板库

## 项目目标

整理一套可复用的 Prompt 模板，覆盖日常学习、编程、写作和结构化信息提取。

## 项目产物

建议最终形成：

```text
prompt-library/
  summary.md
  translation.md
  code-explain.md
  requirement-analysis.md
  json-extraction.md
  sql-generation.md
  test-case-generation.md
```

## 必备模板

### 1. 文章总结

能力要求：

- 一句话总结
- 关键要点
- 适合人群
- 行动建议
- 不编造原文没有的信息

### 2. 代码解释

能力要求：

- 说明代码整体作用
- 解释关键逻辑
- 指出易错点
- 给出简化版本

### 3. 结构化信息提取

能力要求：

- 严格输出 JSON
- 字段缺失用 null
- 不输出解释文本
- 适合程序继续处理

### 4. 需求分析

能力要求：

- 分析目标
- 分析用户场景
- 拆解核心功能
- 识别边界情况
- 识别技术风险

## 迭代方式

1. 先写最简单版本。
2. 用 3-5 个不同输入测试。
3. 记录模型输出问题。
4. 给 Prompt 增加约束或示例。
5. 再次测试。

## 验收标准

- 模板能被重复使用。
- 输出格式稳定。
- 对缺失信息不会编造。
- 对复杂任务有清晰步骤。
- 至少包含 5 个不同场景模板。

## 扩展方向

- 为每个模板增加好例子和坏例子。
- 为 JSON 类模板增加 schema。
- 将模板做成命令行工具。
- 接入 LLM API，形成可运行的小应用。
