# Agent 练习

## 练习 1：区分 Chatbot 和 Agent

判断下面场景更适合 Chatbot 还是 Agent：

1. 解释什么是 Prompt。
2. 自动查资料并写一份报告。
3. 翻译一句英文。
4. 读取项目、修改代码、运行测试。
5. 根据用户目标连续调用多个工具。

## 练习 2：拆解目标

用户目标：

```text
帮我学习 MCP，并整理一份适合初学者的笔记。
```

请拆成 5 个可执行步骤。

## 练习 3：设计 Agent 循环

写出一个 Agent 循环，必须包含：

- goal
- plan
- act
- observe
- reflect
- finish
- max_iterations

## 练习 4：设计工具集

为自动研究助手设计 3 个工具：

- 搜索资料
- 读取文档
- 保存笔记

为每个工具写出：

- 工具名称
- 工具描述
- 输入参数
- 风险等级

## 练习 5：终止条件

假设 Agent 一直搜索资料但没有新发现。

请设计 3 条停止规则，避免无限循环。

## 练习 6：权限判断

判断下面动作是否需要用户确认：

1. 搜索公开资料
2. 读取当前项目 README
3. 修改代码文件
4. 删除文件
5. 提交 git commit
6. push 到远程仓库

## 练习 7：反思机制

写一个反思 Prompt，让 Agent 在每次工具调用后判断：

- 当前结果是否有用
- 是否需要继续
- 下一步应该做什么
