# 00 AI 基础导学

## 学习目标

用最直观的方式理解 AI、机器学习、深度学习、大模型、LLM、Agent、RAG、MCP 和 Claude Code 之间的关系，为后续 01-08 阶段打基础。

这个阶段不要求会深度学习数学，也不要求真实 API Key。你只需要能看懂基本 Python 命令、愿意动手记录和复盘。

## 核心概念

- AI、机器学习、深度学习、大模型
- LLM、Chatbot、Agent
- Prompt、Token、上下文窗口
- 幻觉、温度、结构化输出
- Embedding、RAG、知识库问答
- Tool Use、Function Calling、MCP
- Claude Code、Skill、CLI Agent
- Eval、安全、权限边界

## 学习任务

1. 能用自己的话区分 AI、ML、DL、LLM。
2. 能解释 Chatbot、Agent、RAG、MCP 的区别。
3. 能理解 token、上下文窗口和幻觉的直观含义。
4. 能说出开发中使用 AI 的安全边界。
5. 能画出“用户问题 -> Prompt -> LLM -> 工具/RAG/Agent -> 输出 -> 评测”的流程。

## 建议耗时

- 阅读基础讲解：30-45 分钟
- 完成练习：45-60 分钟
- 完成小项目：1-2 小时

## 学完你应该能做到

- 看到 AI 相关术语不再迷路。
- 知道什么时候用 Prompt、什么时候用 RAG、什么时候需要 Agent。
- 知道为什么不能把 AI 当数据库或权威事实来源。
- 知道后续 01-08 阶段各自解决什么问题。
