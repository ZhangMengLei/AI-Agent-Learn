# AI 基础练习

## 练习 1：画出 AI 概念关系图

请用文本画出下面概念的包含或协作关系：

- AI
- Machine Learning
- Deep Learning
- LLM
- Prompt
- RAG
- Agent
- MCP
- Eval

要求：

1. 至少画出两层关系。
2. 每个概念写一句话解释。
3. 标出你还不确定的概念。

## 练习 2：区分 Chatbot、RAG、Agent

下面需求分别适合 Chatbot、RAG 还是 Agent？请说明原因。

1. “解释 Python 中 list 和 tuple 的区别。”
2. “根据公司内部报销制度回答员工问题，并给出处。”
3. “检查一个 PR，运行测试，发现失败后尝试修复。”
4. “把用户输入转换为固定 JSON。”
5. “每天读取构建结果，失败时创建 issue。”

## 练习 3：识别幻觉风险

阅读下面回答，指出可能的幻觉风险：

```text
这个库一定支持 auto_magic_config 参数，你只要在配置里打开它就能解决所有问题。
```

请回答：

1. 哪些词说明它可能不可靠？
2. 你会如何验证？
3. 如何改写 Prompt 让模型更谨慎？

## 练习 4：设计一个好 Prompt

任务：让 AI 帮你解释一段陌生代码。

请写一个 Prompt，必须包含：

- 角色
- 任务
- 背景
- 输出格式
- 不确定时的处理方式
- 验收标准

## 练习 5：AI 使用安全判断

判断下面行为是否安全，并说明原因：

1. 把 `.env` 文件贴给 AI，让它帮忙排查连接失败。
2. 把脱敏后的错误日志贴给 AI。
3. 让 AI 生成单元测试草稿。
4. 让 Agent 自动执行 `git push --force`。
5. 让 AI 根据公开 README 写快速开始文档。
