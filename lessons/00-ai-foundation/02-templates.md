# AI 基础常用模板

本阶段的模板用于帮助小白把抽象概念问清楚、画清楚、复述清楚。

## 模板 1：概念解释

```text
你是 AI 入门老师。请用小白能懂的方式解释：<概念>。

要求：
1. 先用一句话说明它是什么。
2. 用生活类比解释。
3. 说明它在 AI 应用中的作用。
4. 给一个简单例子。
5. 列出 3 个常见误区。
6. 如果涉及不确定或高级细节，请标记为“进阶再学”。
```

## 模板 2：概念对比

```text
请比较下面几个概念：<概念 A>、<概念 B>、<概念 C>。

请用表格输出：
- 解决什么问题
- 输入是什么
- 输出是什么
- 适合什么场景
- 不适合什么场景
- 小白容易误解的点
```

## 模板 3：学习路线规划

```text
我是 AI 初学者，背景是：<你的技术背景>。
我想达到的目标是：<目标>。
每天可学习时间：<时间>。

请帮我制定 4 周学习路线，要求：
1. 每周目标明确。
2. 每天有小任务。
3. 每周有可运行或可展示产物。
4. 标出哪些内容先了解即可，哪些必须掌握。
```

## 模板 4：把术语翻译成开发语言

```text
请把下面 AI 术语翻译成软件开发者容易理解的说法：<术语列表>。

每个术语请输出：
1. 一句话解释
2. 类似传统开发中的什么概念
3. 它不是传统开发中的什么
4. 一个开发场景例子
```

## 模板 5：判断该用什么 AI 方案

```text
我有一个需求：<需求描述>。

请判断更适合使用 Chatbot、Prompt、RAG、Tool Use、Agent、MCP 还是 Claude Code。

请输出：
1. 推荐方案
2. 为什么
3. 不推荐哪些方案
4. 最小实现路径
5. 风险和安全边界
```
