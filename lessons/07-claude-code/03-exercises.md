# Claude Code 练习

## 练习 1：识别适合 CLI Agent 的任务

判断下面任务是否适合 Claude Code：

1. 解释一个函数作用
2. 修改项目代码并运行测试
3. 删除远程数据库
4. 生成 PR 总结
5. 批量重构变量名
6. 给线上用户发通知

说明哪些需要人工确认。

## 练习 2：编写 CLAUDE.md

为一个项目写 CLAUDE.md，至少包含：

- 项目简介
- 常用命令
- 测试方式
- 代码规范
- 安全注意事项

## 练习 3：设计 Slash Command

设计一个 `/study-next` 命令，用于学习下一个 AI Agent 主题。

说明：

- 输入是什么
- 执行步骤是什么
- 输出格式是什么

## 练习 4：设计 Hook

设计一个 hook：当文件被修改后自动运行格式化。

思考：

1. 什么时候触发？
2. 执行什么命令？
3. 命令失败时怎么办？

## 练习 5：权限分类

把下面操作分为允许、需要确认、禁止自动执行：

- 读取 README
- 运行单元测试
- 安装依赖
- 删除文件
- git commit
- git push
- force push
- 修改 CI 配置

## 练习 6：子 Agent 任务拆分

如果要生成一整套学习文档，可以如何拆给多个子 Agent？

请按主题拆分，并说明每个子 Agent 的职责。
