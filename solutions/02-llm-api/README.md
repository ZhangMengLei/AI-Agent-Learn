## 建议使用顺序

1. 先完成课程练习：[lessons/02-llm-api/03-exercises.md](../../lessons/02-llm-api/03-exercises.md)
2. 再查看：[03-exercises.md](./03-exercises.md)
3. 做练习工程后查看：[04-exercises-lab.md](./04-exercises-lab.md)
4. 阅读项目说明后查看：[05-project.md](./05-project.md)
5. 实现项目工程后查看：[06-project-lab.md](./06-project-lab.md)

- 可运行实现：当前阶段以设计和文档练习为主。

## 判断自己是否过关

- 能用自己的话解释本阶段核心概念。
- 能说清楚输入、输出、边界和风险。
- 能把答案中的参考方案改造成自己的实现。
- 能说明为什么某些操作需要权限控制或测试验证。
