# Solutions 使用说明

本目录用于放置课程练习和项目的参考答案索引。参考答案不建议一次性写满，也不建议学习者先看答案再做练习。

## 使用原则

1. 先独立完成 `lessons/` 中的练习和项目。
2. 再使用 `checkpoints/` 做自查，标出不确定的问题。
3. 最后再查看对应参考答案或讲解。
4. 参考答案只作为一种实现方式，不代表唯一正确答案。
5. 如果你的方案更简单、更安全、更容易测试，可以优先保留自己的方案。

## 索引规划

后续可按阶段逐步补充：

```text
solutions/
  README.md                  当前说明文件
  01-prompt/                 Prompt 练习参考
  02-llm-api/                LLM API 练习参考
  03-tool-use/               工具调用练习参考
  04-rag/                    RAG 项目参考
  05-agent/                  Agent 项目参考
  06-mcp/                    MCP 项目参考
  07-claude-code/            Claude Code 工作流参考
  08-eval-security/          评测与安全参考
```

## 建议答案格式

每个参考答案建议包含：

- 题目链接。
- 解题思路。
- 关键代码或关键 Prompt。
- 如何运行。
- 如何测试。
- 常见错误。
- 可选扩展。

当前文件只提供索引和使用说明，不展开完整答案。
