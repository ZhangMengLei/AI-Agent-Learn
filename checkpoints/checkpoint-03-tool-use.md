# Checkpoint 03：Tool Use / Function Calling

## 概念自查

1. Tool Use 和普通文本生成有什么区别？
2. 工具名称、描述、参数 schema 为什么会影响模型选择工具的准确性？
3. 工具调用结果应该原样返回给模型，还是需要做结构化处理？
4. 哪些工具属于高风险工具，需要权限确认？
5. 如何避免模型为了完成任务而调用不合适的工具？

## 实践自查

1. 能否设计一个只读工具和一个写入工具？
2. 能否为工具参数设置必填字段、类型和约束？
3. 能否处理工具执行失败、空结果和格式错误？
4. 能否让模型根据用户意图选择是否调用工具？
5. 能否记录每次工具调用的输入、输出和耗时？

## 工程自查

1. 是否为危险工具增加了人工确认或白名单？
2. 是否避免把工具错误直接暴露成误导性回答？
3. 是否限制了工具可访问的数据范围？
4. 是否有工具调用的单元测试或模拟实现？
5. 是否能从日志中复盘一次完整工具调用链路？
