# Checkpoints 自查清单

本目录用于在每个阶段结束时做自查。建议学习顺序：先完成对应 `lessons/` 阶段的基础讲解、模板、练习和项目，再打开相应 checkpoint 回答问题。

## 使用方式

1. 不看参考答案，先独立回答自查问题。
2. 把回答写到自己的学习笔记中，标注“不确定”的地方。
3. 回到对应课程文档或 demo 复习薄弱点。
4. 如果某一阶段超过三分之一问题答不清楚，建议不要急着进入下一阶段。
5. 完成项目后，再用 checkpoint 复查一次工程边界、测试和安全。

## 推荐通过标准

- 能用自己的话解释核心概念。
- 能独立完成最小 Demo。
- 能说清楚输入、输出、异常和边界。
- 能说明如何测试、评估和排查问题。
- 不在代码、日志、样例数据中暴露真实密钥。
