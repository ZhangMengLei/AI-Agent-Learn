# Agent 和 Chatbot

## Chatbot 是什么

Chatbot 通常是围绕对话输入和回答输出构建的程序。它擅长回答问题、解释概念、总结文本，并在单轮或多轮对话中保持上下文。

Chatbot 的主要目标是把用户问题转成自然语言答案。它一般不会主动拆解长期目标，也不会持续调用多个工具完成复杂任务。

## Agent 是什么

Agent 是围绕目标、规划、工具调用和观察反馈构建的系统。它不仅生成文字，还可以根据目标选择工具、执行步骤、读取结果，并继续决定下一步。

Agent 更像一个带有工作循环的执行者：理解目标，制定计划，调用工具，观察结果，修正计划，直到任务完成或需要人类确认。

## 核心区别

Chatbot 偏向回答问题，Agent 偏向完成任务。Chatbot 的输出通常是一次回答；Agent 的输出可能来自多步行动和工具结果。

当需求只是解释概念或闲聊时，Chatbot 足够。当需求包含搜索、写文件、运行命令、检查结果、迭代修复时，更适合使用 Agent。
